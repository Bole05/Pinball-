# Soccer Pin Ball

## Description

Lanza la pelota a la portería y obten la máxima puntuación. Usa los flippers para lanzar la pelota.

## Key Features

 - F1 : Introducir una pelota al juego
 - F2 : Introducir un rectangulo
 
## Controls

- Teclas para mover los flippers: <- y ->
